## 🚀 Getting Started

### 1. **Clone or Download**
```bash
git clone <repository-url>
cd visa-guide-new
```

### 2. **Open in Browser**
```bash
# Simply open index.html in any modern web browser
# No build process or server installation needed
```

### 3. **Or Run with Local Server**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server
```

Then navigate to `http://localhost:8000`

## 📖 How to Use

### Search Functionality
1. Enter a country name, region, or visa type in the search bar
2. Results update in real-time
3. Click "Learn More" on any country for details

### Browse by Region
1. Scroll to "Explore by Region" section
2. Click on any region card
3. View all countries in that region

### Find Visa Types
1. Check "Popular Visa Types" section
2. Learn about different visa categories
3. See which countries offer each type
