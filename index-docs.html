How to Get Started

Option 1: Open Directly

Simply double-click index.html to open the website in your browser.

Option 2: Local Server

For better functionality, run a local server: 

cd /workspaces/visa-guide-new
python -m http.server 8000
# Then open: http://localhost:8000
Note: The website works perfectly without any build process or installation. Just open the HTML files!
